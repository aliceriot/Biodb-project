    FAQ

    What is CSB and how does it work?
    CSB (Conserved Sequence blocks)= conserved regions in the sequences(CSB1, CSB2, CSB3). To restrict the result with CSB, simply check the box for the desired CSB. The system provides three possible options for your output data: csb1, csb2, and csb3.
    In our given definition, csb1 = atgggcgtgc, csb2 = ccccgtgc, csb3 = ggggttggtgta.

    How do I run the display graphic represenation?
    Our display page uses Jalview to show the sequence alignments. You need a computer that can run Java 1.6 or later to run the full Jalview application. Click here to install. It doesn't work on Chrome 32-bit browser on Mac OS X. Try firefox when it happens.

    How can I filter my data by cluster information?
    The cluster information is only available with the sequences of CSB3. You will need to first check "CSB3" box to see the search box for cluster related filters.
